тут будем писать основные команды гит

команда git init создает в корневом каталоге проекта репозиторий

git -- version - версия git git status – посмотреть текущее состояние, что добавлено, что не добавлено. git add index.html – добавить изменение в репозиторий. git add . – если ставим точку, то тогда будет все, что добавлено все что было. git log - история коммитов git clone - клонируем репозиторий
